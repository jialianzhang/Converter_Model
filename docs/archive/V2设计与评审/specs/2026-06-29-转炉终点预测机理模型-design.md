# 基于耦合反应机理的转炉冶炼终点预测模型 — 设计文档 (V2)

## 1. 项目目标

基于张强(2024)和Kitamura(2009)的耦合反应机理模型，用Python语言复现转炉冶炼过程模型，实现终点钢水碳含量(C)和终点钢水温度(T)的预测。V2版本在V1基础上进行了架构重构：引入**混合能驱动的动态传质系数**机制，修正了界面氧平衡方程的化学计量数错误，升级了数值求解器和终止条件。

**参考基准（张强原文命中率）：**
- 终点碳含量 ±0.02% 命中率：89%
- 终点钢水温度 ±10℃ 命中率：83.0%

**V2 核心升级点：**
- **动态传质系数**：k_m和k_s不再固定，每时间步由混合能和固相率动态更新
- **公式纠错**：界面氧平衡方程(Mn/P/V/Ti)化学计量数修正为标准氧化态
- **E_M动态化**：表观平衡常数在Newton求解器内部实时更新
- **求解器升级**：Brent有界寻根为主 + Newton-Raphson为辅，防负数崩溃
- **自适应时间步长**：1s(剧烈脱碳)/2s(常规)/5s(末期)
- **复合终止条件**：达到最低吹氧量 且 终点C满足目标

**参考论文：**
- 张强 等 (2024) — 基于耦合反应机理的转炉冶炼模型（主要参考）
- Kitamura 等 (2009) — Analysis of Dephosphorization Reaction Using a Simulation Model of Hot Metal Dephosphorization by Multiphase Slag（耦合反应动力学基础，张强原文参考文献[16]）
- 徐建飞 等 (2016) — 钢中硅质量分数对出钢渣洗脱硫效果影响（耦合反应动力学参考）

## 3. 13步主循环 (The 13-Step Main Loop)

V2版采用严格的单步时域离散(Euler-forward)顺序，每个时间步按以下顺序执行：

```
Step  1: 更新混合能     (计算 ε_T, ε_B → ε_total)
Step  2: 更新修正传质系数 (用上一步 q_CO_prev 计算 k_s, G_CO, 更新全部 F_i)
Step  3: 计算并更新表观平衡常数 E_M (受当前T和活度影响)
Step  4: 求解界面氧活度 a*_O (Brent/Newton 寻根)
Step  5: 计算界面浓度 w([M])*, w((MOn))* (代入 a*_O)
Step  6: 计算反应速率 Δm_i/Δt (得到当前步 q_CO，供下一时间步使用)
Step  7: 更新钢液相质量与成分
Step  8: 更新炉渣相质量与成分
Step  9: 更新石灰等造渣料的溶解状态
Step 10: 更新废钢的熔化状态
Step 11: 更新热量平衡(统筹各项收支)
Step 12: 更新温度 T^(l+1) = T^(l) + ΔT
Step 13: 更新热力学参数(为下一时间步准备)
```

**终止条件（复合）：** `达到最低目标吹氧量下限 AND w([C]) ≤ target_C_end`

**自适应时间步长：** `1s`(脱碳剧烈期, w([C])>0.8%) / `2s`(常规平稳期, 0.3%<w([C])≤0.8%) / `5s`(吹炼末期, w([C])≤0.3%)

**时序代数环处理（重要）：** Step 2计算k_s和G_CO时需要的q_CO由脱碳速率J_C决定，而J_C要到Step 6才能算出。这是一个典型的前向欧拉时序问题——**Step 1和Step 2必须使用上一时间步(t−Δt)的q_CO_prev来计算当前的混合能和传质系数**。初始化：t=0时设定q_CO_prev = 0。

### 4.2 混合能与动态传质模块（`mixing_energy.py` + `modified_mass_transfer.py`）

这是V2最核心的新增模块。k_m和k_s不再是定值，而是每时间步根据流体动力学状态动态更新。

#### 4.2.1 混合能计算

**顶吹混合能 ε_T（式14）：**

ε_T = 0.0403 · Q_T · D · U² · cos²θ / (W · X)

- Q_T: 顶吹氧气流量 (m³/min)
- D: 喷枪喷嘴直径 (m)
- U: 喷枪喷嘴出口气体线速度 (m/s)
- θ: 喷嘴与氧枪垂直轴的夹角 (°)
- W: 铁水质量 (kg)
- X: 枪位高度 (m)

**底吹混合能 ε_B（式15）：**

ε_B = (28.5 · Q_B · T / W) · lg(1 + H/148)

- Q_B: 底吹氩气流量 (m³/min)
- T: 当前铁水温度 (K)
- H: 熔池深度 (m)

**总混合能：** ε_total = ε_T + ε_B

#### 4.2.2 动态传质系数更新

**钢液相物理传质系数 k_m：** 钢液传质主要受整体搅拌强度影响，与总混合能的相对变化率呈指数关系：

k_m = k_m⁰ × (ε_total / ε_total⁰)^γ

- k_m⁰: 基准钢液传质系数（由Kitamura(2009)标定值或参数标定获得）
- ε_total⁰: 基准状态总混合能
- γ: 搅拌指数（0.2~0.5）

**炉渣相物理传质系数 k_s：** 炉渣传质受CO气体搅拌和固相率双重影响：

第一步 — 气体搅拌与渣量影响：
k_s' = k_s⁰ × ((q_Ar + 3·q_CO) / q_Ar⁰) × (W_s⁰ / W_s)

第二步 — 固相率衰减：
k_s = k_s' × 10^(-n · f_solid)

- k_s⁰: 基准炉渣传质系数
- q_Ar: 底吹氩气流量 (Nm³/s)
- q_CO: 当前时间步脱碳生成的CO流量 (Nm³/s)
- W_s: 当前炉渣总质量 (kg)
- f_solid: 炉渣固相率 (0~1)
- n: 衰减常数，Kitamura标定值为5

**数值保护：** 当f_solid逼近1.0时，k_s会缩小十万倍以上。需在代码中为k_s设定极小值下限：`k_s = max(k_s, 1e-8)`，防止修正传质系数F_i趋于零导致除零错误或反应完全停滞。

**注：** k_m^(1/2)关系来自Kitamura原文式(6)：k_m = k_m⁰·(q_Ar/q_Ar⁰)^(1/2)。此处将其推广为以总混合能为自变量的更一般形式(γ=0.5时退化为原关系)。

#### 4.2.3 换算为修正传质系数F_i

**钢液侧各元素：** F_M = k_m · ρ_m / (100 · M_M)

**炉渣侧各氧化物：** F_MOn = k_s · ρ_s / (100 · M_MOn)

- ρ_m: 钢液密度 ≈7000 kg/m³
- ρ_s: 炉渣密度 ≈3500 kg/m³
- M_M: 元素M的摩尔质量 (kg/mol)
- M_MOn: 氧化物MOn的摩尔质量 (kg/mol)

#### 4.2.4 反应界面面积A的动态更新

依据Kitamura(2009)，界面面积A随渣中液相比例变化：
A ∝ (液相分数)^(2/3)

即：A = A⁰ × (1 − f_solid)^(2/3)

### 4.3 耦合反应求解器（拆分为3个子模块）

#### 4.3.1 界面浓度计算 (`model/interface.py`)

**界面质量分数通用公式（式7，适用于所有元素C,Si,Mn,P,V,Ti,Fe）：**

w([M])* = (F_MOn · w((MOn))b + F_M · w([M])b) / (F_M + F_MOn · E_M · (a*_O)^n)

#### 4.3.2 氧平衡残差 (`model/oxygen_balance.py`)

**修改后的界面氧平衡方程（V2关键修正）：**

Residual(a*_O) = 2J_Si + J_Mn + 2.5J_P + 1.5J_V + 2J_Ti + J_Fe + J_C − J_O = 0

**各元素氧化反应n值（按标准氧化态修正后）：**

| 元素 | n值 | 氧化产物 | 系数来源 |
|------|-----|---------|---------|
| Si | 2 | SiO₂ | 标准 |
| Mn | **1** | MnO | **修正**(原张强1.5→标准1) |
| P | **2.5** | P₂O₅ | **修正**(原张强2→标准2.5) |
| V | **1.5** | V₂O₃ | **修正**(原张强1→1.5) |
| Ti | **2** | TiO₂ | **修正**(原张强1→标准2) |
| Fe | 1 | FeO | 标准 |
| C | 1 | CO | 标准 |

**关键说明：**
- 张强原文的n值（Mn=1.5, P=2, V=1, Ti=1）是非标准简化值，V2按标准氧化态修正以保证物质守恒的严密性
- 对应地，各元素在式(3)中的表观平衡常数E_M所用K_M也必须按修正后的氧化产物查找对应的热力学数据

**J_O定义：** J_O = F_O (w([O])b − w([O])*) = F_O/f_O (a^b_O − a*_O)

#### 4.3.3 寻根算法 (`model/root_solver.py`)

```
算法流程：
  ┌─ 以Brent法为主求解器 ──────────────────┐
  │ 1. 设定有界区间: a*_O ∈ [1e-8, 1.0]    │
  │ 2. Brent迭代:                          │
  │    a. Guess a*_O                       │
  │    b. 更新当前E_M (受T和成分影响)        │
  │    c. 计算各元素界面浓度 w([M])*        │
  │    d. 计算各元素J_M                     │
  │    e. 评估残差 Residual(a*_O)           │
  │    f. 调整a*_O                         │
  │ 3. 收敛判据: |Residual| < 1e-10         │
  │ 4. 若Brent失败→回退Newton-Raphson      │
  │    (NR发散→回退二分法)                  │
  └────────────────────────────────────────┘
```

**E_M动态更新（重要）：** E_M不是常数。式(3)中的E_M = 100·C·N_MOn·f_M·K_M/(ρ_S·f_MOn)中，f_M、f_MOn、C均随时间和成分变化。必须在求解器的每次迭代循环内部重新计算E_M，不可置于循环外。

**式(5)脱碳逻辑澄清：** J_C = F_C(w[C]^b − w[C]*) = G_CO(P*_CO − 1) 是**速率平衡方程**（传质速率=CO生成速率），不可将其与经验公式式(16)混用。模型中废弃经验公式式(16)，G_CO同样受混合能动态控制：

G_CO = G_CO⁰ × ((q_Ar + 3·q_CO) / q_Ar⁰) × (W_s⁰ / W_s)

### 4.4 石灰熔化模型 (`model/lime_melting.py`)

沿用Kitamura(2009)的石灰溶解模型（张强原文亦引用于此）：

**传质系数（Kitamura式11）：** k' = 0.35 · ρ_s^(0.36) · η^(−0.36) · D^(0.67) · d^(−0.31) · U^(0.67)

- ρ_s: 炉渣密度 (kg/m³)；η: 炉渣黏度 (Pa·s)；D: CaO在渣中扩散系数 (m²/s)
- d: 石灰粒径 (m)；U: 石灰与渣相对速度，取0.5 m/s

**质量变化率（Kitamura式12）：** −dW_CaO/dt = n_CaO · A_CaO · {(k'·ρ_s)/(100·r')} · Δ(%CaO) · r' · x

- Δ(%CaO): 石灰浓度与饱和浓度的差值
- x: 表面积拟合参数（工业石灰多孔性修正）

**C₂S饱和区特殊处理：** 当渣成分进入C₂S饱和区域（固液共存）时，k'减小至1/10（C₂S在石灰表面形成致密层阻碍溶解）。

**CaO饱和浓度：** CaO-SiO₂-FeO系相图C₂S区域简化模型→查表+插值。熔解条件同V1：r < 1mm → 完全熔解。

### 4.5 废钢熔化模型 (`model/scrap_melting.py`)

- 废钢熔点（式11）：T_MP = 1809 − 8.13·w(C)² − 54·w(C) (K)
- T_bath < T_MP → 扩散熔化（碳浓度梯度驱动）
- T_bath ≥ T_MP → 强制熔化（温差驱动）

### 4.6 物质/热量平衡 (`model/mass_balance.py`, `model/heat_balance.py`)

- 物质平衡（式12）：m_i^l = m_i^(l-1) − Δm_i + ΣΔm_i^CM
- 温度更新（式13）：ΔT/Δt = (1/(c_M·m_M+c_S·m_S)) · ((Q_HG−Q_HC−Q_CM)/Δt)

热收入项：
- Σ元素氧化放热（按修正后氧化产物重新核算放热量）
- C₂S(2CaO·SiO₂)形成热
- C₃P(3CaO·P₂O₅)形成热（按P₂O₅核算）
- 二次燃烧热(CO→CO₂)

热支出项：
- 造渣料熔化/分解吸热（石灰、白云石、烧结矿）
- 废钢熔化吸热
- 烟气带走热 + 水分蒸发吸热 + 碳酸盐分解吸热
- 辐射热损失 + 粉尘热损失 + 炉衬吸热

### 4.8 参数标定 (`calibration/optimizer.py`)

**V2标定策略（物理意义保留）：**

不直接暴力优化k_m、k_s的数值，而是以Kitamura(2009)提供的经典参数为基准，优化**修正乘子(α)**。

| 参数 | 符号 | 单位 | 基准来源 | 标定方式 |
|------|------|------|---------|---------|
| 钢液传质系数修正乘子 | α_steel | - | Kitamura基准 | 优化α_steel: k_m⁰ = α_steel × k_Kitamura_base |
| 炉渣传质系数修正乘子 | α_slag | - | Kitamura基准 | 优化α_slag: k_s⁰ = α_slag × k_Kitamura_base |
| CO生成速率基准 | G_CO⁰ | mol/(s·m²) | Kitamura | 优化G_CO⁰ |
| 搅拌指数 | γ | - | 0.2~0.5 | 优化γ |
| 二次燃烧比例 | η_post | - | 0.10 | 标定 |
| 辐射热损失系数 | α_rad | - | 0.03 | 标定 |
| 粉尘损失比例 | ε_dust | - | 0.01 | 标定 |

**目标函数：** (1/N) Σ [w_C·(C_pred_i − C_actual_i)² + w_T·(T_pred_i − T_actual_i)²]

**优化方法：** scipy.optimize.minimize，L-BFGS-B（有梯度）或 Nelder-Mead（无梯度）

**数据划分：** 训练集70% / 验证集30%

**输出：** 标定参数表、训练/验证集命中率、C_pred vs C_actual散点图、T_pred vs T_actual散点图

## 8. 活度模型说明

### 8.1 金属相活度 — Wagner模型 (`thermo/wagner.py`)

lg f_i = Σ e_i^j · w([j])

采用文献中的Wagner相互作用系数e_i^j(1873K)，温度修正为：
e_i^j(T) = (1873/T) · e_i^j(1873K)

### 8.2 炉渣相活度 — 离子结构理论 (`thermo/ion_theory.py`)

f_iOn = 1 / Σ_j [ w((jOn))b_L · exp(−ε_ij/RT) ]

其中ε_ij为两种不同元素的交换电子能(J/mol)，取自文献值。

若离子结构理论关键参数缺失，此模块预留**光学碱度模型**作为Fallback方案。
