# 论文素材 03 · 参数标定与泛化性

## 【文字】标定的性质（回应"标定参数是否只适用一炉"）

标定的对象是一组**设备/工艺级常数**（二次燃烧率、综合散热、乳化面积、传质基准等），由 N 炉数据联合优化；炉次之间的差异（铁水成分、温度、废钢量、供氧量）由每炉输入变量承载，机理公式负责把不同输入映射到不同终点。这是机理模型对黑箱模型的本质优势：炉次差异走输入、设备特性走参数，故一组参数对新炉次仍有预测力。泛化性由 70/30 训练/验证划分实证：验证集命中率≈训练集 ⇒ 参数抓住的是设备特性而非炉次噪声。需要重标的情形：换炉役（炉衬侵蚀改变散热/熔池几何）、换喷头、原料结构大变、换转炉。

## 【文字】标定方法

目标函数为 N 炉加权归一均方误差（尺度：ΔT/10°C、ΔMn/0.02%、ΔP/0.005%），**框架 = 目标碳终止（使用框架，与张强同口径）**：碳按终止设定收口（构造性命中，不入目标函数 w_C=0），温度/锰/磷为自由预测入目标（权重 1.0/0.5/0.3）。出钢量不入目标（含未建模喷溅损失，作目标会推高铁氧化）仅作报告；TSO 终点氧为独立验证指标（模型 a_O* 为界面值、与本体溶解氧有系统偏置，不可直接作目标）。优化器 Nelder-Mead 有界（scipy），100 炉随机 70/30 划分（固定种子），18 进程并行，~16 分钟收敛。

## 【结果】泛化性（论文关键表，V1.0 冻结参数实算）

| 指标 | 训练集(70炉) | 验证集(30炉) |
|------|-------------|-------------|
| 温度 T±10°C 命中率 | 28.6% | **30.0%** |
| 温度 MAE | 28.2°C | **25.0°C** |
| 锰 Mn±0.02% 命中率 | 68.6% | **73.3%** |
| 锰 MAE | 0.0169% | 0.0162% |
| 磷 MAE | 0.0111% | 0.0112% |
| 碳 C±0.02%（构造性） | 100% | 100% |

**验证集与训练集几乎一致（温度 MAE 与命中率甚至更优）——参数未过拟合，抓住的是设备级共性。** 这是"一组参数打全部炉次"的直接实证。

（同源性说明：本表为冻结参数（config 默认值，舍入至 5 位小数）的实算结果，由 `_paper_stats.py` 一键复现（data/results/paper_stats_v1.txt）。标定报告 final_calibration_v1.txt 使用优化器全精度参数评估，与冻结值存在 ≤0.4°C 的 MAE 漂移（含钳位与变步长的仿真对参数尾数敏感）；论文一律引用冻结实算值，勿混用两套数字。）

## 【结果】标定数据集画像

~135t 转炉，100 炉：铁水 127.8~145t、1245~1380°C，C 3.90~4.54%、Si 0.24~0.70%、P 0.080~0.155%；废钢比 18~27%（大废钢比操作）；供氧 562~729 m³/min；实测耗氧 6906~8397 Nm³；终点 C 0.031~0.078%、T 1582~1678°C、TSO 氧 431~926 ppm。数据自洽性：耗氧/流量×60 与吹炼时长逐炉吻合；碳氧积 [C][O]=0.0025~0.0045 符合碳氧平衡规律（TSO 数据质量的直接证据）。

## 【写作提示】

- eta_O2_late 固定与 aO_cap_CO_mult 标定的"定值依据"务必写明（工业渣 FeO 范围 / TSO 氧与碳氧平衡关系）——这是"参数有物理身份"论证的核心。
- scrap_heat_scale 触下界 0.60：如实说明其吸收了废钢比相关的结构残差（见素材 05 局限），是参数吸收模型简化的例证，不掩饰。
- 若审稿人问"为何不用 L-BFGS-B"：仿真含分段/钳位逻辑，数值梯度不稳，Nelder-Mead 有界更稳健。
