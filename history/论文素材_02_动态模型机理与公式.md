# 论文素材 02 · 动态模型机理与公式

## 【文字】总体流程

动态模型按 13 步 Euler-forward 循环推进（自适应步长：w(C)>0.8% 取 1s、0.3~0.8% 取 2s、≤0.3% 取 5s）：每步先更新混合能与传质系数，再解渣-金界面耦合反应（单一界面氧活度 a_O*），施加反应通量与物质平衡，处理辅料溶解、废钢熔化，最后按符号化热平衡更新温度。终止方式两种：**目标碳终止**（吹到 w(C)≤目标停，使用/张强口径）与**氧锚定终止**（吹完实际耗氧量读终点，检验口径）。开吹含 60s 点火爬坡（O₂ 流量线性升至额定）。

## 【公式】混合能与动态传质（Nakanishi + Kitamura 扩展）

$$\varepsilon_T = \frac{0.0403\, Q_T D U^2 \cos^2\theta}{W X}, \qquad \varepsilon_B = \frac{28.5\, Q_B T}{W}\lg\left(1+\frac{H}{148}\right)$$

$$k_m = \alpha_{steel} k_m^0 \left(\frac{\varepsilon_{total}}{\varepsilon_{total}^0}\right)^{\gamma}, \qquad k_s = \alpha_{slag} k_s^0 \cdot \min\!\left(1+\frac{3q_{CO}}{q_{Ar}},10\right)\frac{W_{ref}}{m_{slag}} \cdot 10^{-n f_{solid}}$$

$$F_M = \frac{k_m \rho_m}{100 M_M}, \quad F_{MO_n} = \frac{k_s \rho_s}{100 M_{MO_n}}, \quad A = (A_0\!\times\!75 + A_{em})(1-f_{solid})^{2/3}$$

（q_CO 用上一时间步值，前向欧拉破代数环；k_s ≥ 1e-8 数值保护；A_em 为乳化液滴有效面积，标定参数。）

## 【文字+公式】双路径氧化与氧账本

**路径一（火点直接氧化）**：O₂ 按份额分配——直接脱碳 C_direct（标定 ~0.25）、Si 0.08、Mn 0.02（带渣金平衡门控，见下）、逸散 0.05，**其余全部 [Fe]+½O₂→(FeO)**——间接氧化理论：火点暴露的主要是 Fe，多数氧先生成 FeO 入渣。末期"入铁氧保留率" retain = η_late + (1−η_late)·min(w_C/1, 1)：碳低时火点碳饥饿，过剩入铁氧按 (1−retain) 逸出而非无限生成 FeO。**氧账本每步记账**（吹入 = 直接氧化 + 界面注入 + 逸散），全炉化学利用率均值 89.3%（100 炉，V1.0 冻结参数；名义逸散 5% + 末期入铁氧通道过剩氧逸出）。

**路径二（渣-金界面耦合反应）**：C/Si/Mn(/V/Ti) 在界面竞争氧，氧源为渣中 FeO 分解与火点溶解氧注入。

## 【公式】耦合反应求解器（核心）

界面浓度通用式（Si/Mn/P/V/Ti；张强式 7）与碳专属式（CO 气体产物，含 f_C）：

$$w([M])^* = \frac{F_{MO_n} w((MO_n))^b + F_M w([M])^b}{F_M + F_{MO_n} E_M (a_O^*)^n}, \qquad w([C])^* = \frac{F_C w([C])^b + G_{CO}}{F_C + G_{CO} K_C f_C\, a_O^*}$$

$$E_M = \frac{100\, C_{slag} N_{MO_n} f_M K_M}{\rho_S f_{MO_n}} \quad (N_{MO_n}\ \text{取半氧化物单元 PO}_{2.5}/\text{VO}_{1.5};\ C_{slag}=\rho_s/\bar M\ \text{实时计算})$$

**Fe 作溶剂（本文对通用式的关键修正）**：K_Fe 对应 Fe(l)+[O]=(FeO)，a_Fe=1，不得代入式(7)（否则等效乘以 w_Fe≈95）。FeO 界面通量由渣侧传质控制：

$$w(\mathrm{FeO})^*_{eq} = \frac{100 K_{Fe}\, a_O^* M_{FeO}}{\gamma_{FeO} \bar M}, \qquad J_{FeO} = F_{FeO}\left[w(\mathrm{FeO})^b - w(\mathrm{FeO})^*_{eq}\right] \le \frac{n_{FeO}^{stock}}{A\,\Delta t}$$

氧平衡残差（Brent 有界求根，需求单调增/供给单调减 → 唯一根）：

$$2J_{Si} + J_{Mn} + 1.5J_V + 2J_{Ti} + J_C \;=\; J_{FeO} + J_{O_2}^{inj}$$

## 【公式】脱磷（Kitamura 式 1 + Suito-Inoue 分配比）

主吹期界面 a_O* 被碳钉低，纯质量作用式下脱磷必然反向；真实脱磷发生在氧势接近渣侧的乳化液滴上。故 P 用速率方程 + 经验分配比（氧由渣中 FeO 显式供给，2[P]+5(FeO)+3(CaO)→(3CaO·P₂O₅)+5[Fe]，质量与热量守恒记账）：

$$-\frac{d[\%P]}{dt} = \frac{A}{V}\cdot\frac{[\%P] - (\%P)/L_P}{1/F_P + 1/(L_P F_{P,slag})}, \qquad \lg L_P = 0.072\left[(\%\mathrm{CaO})+0.3(\%\mathrm{MgO})+\ldots\right] + 2.5\lg(\%T.Fe) + \frac{11570}{T} - 10.52$$

## 【公式】锰的碳氧缓冲平衡（本文机理贡献之一）

末期熔池尚有碳时，本体氧被 C-O 反应缓冲，锰应与**缓冲后的本体氧**平衡而非与界面峰值氧平衡：

$$a_O^{bulk} = \min\!\left(\frac{k_{CO}}{K_C f_C\, w([C])^b},\; a_O^*\right), \qquad w([Mn])_{floor} = \frac{w((\mathrm{MnO}))^b}{E_{Mn}\, a_O^{bulk}}$$

实施于两处：火点 Mn 通道门控（w(Mn)≤floor 时关闭——烧掉即被渣还回，净通道无效）；耦合界面 w([Mn])* 取 max(式7 值, floor)。k_CO 为标定参数（终点实测本体氧 ≈1~1.8× 碳氧平衡）。**该修正与全局氧平衡解耦**——验证显示 Mn 0.030→0.081%（≈实测）而 FeO/温度纹丝不动；反例：直接压全局 a_O* 会掐断 FeO 间接氧化主脱碳路，FeO/温度爆炸（31→81%、1628→2103°C），文中可作耦合强度的论证。

## 【公式】辅料、废钢与热平衡

- 石灰/白云石按实际批次（0/3/6 min）入炉；石灰溶解用 Kitamura 式 11/12（渣黏度换 **Riboud(1981)** 模型 + Roscoe 固相修正，取代无出处自造式）；C₂S 饱和区 k′/10。
- 废钢：质量按抛物线闭合（唯象，明示）；热汇拆分为未熔废钢升温显热一阶弛豫（τ 标定）+ 熔化时补足显热差与潜热——前期冷却效应出现在正确时刻（0-1min 温度微降）。
- 热平衡**符号化**：氧化 +ΔH、还原 −ΔH（FeO 循环：铁氧化 +240、FeO 被碳还原 −240、碳成 CO +140 → 净 +140 kJ/mol = 碳直接氧化，能量自洽，热账本审计通过）；ΔH 取统一基准（溶解元素 + O₂(g) → 渣态氧化物，炼钢温度近似）。
- 温度更新 $\Delta T = (Q_{HG} - Q_{HC}) / (c_M m_M + c_S m_S)$，热账本 11 项逐步累计输出。

## 【公式】活度与热力学数据

- 钢液：Wagner，$\lg f_i = \sum_j e_i^j w[j]$，e 取 Sigworth–Elliott(1873K)，温度修正 (1873/T)。
- 渣：γ_FeO/γ_MnO 用文献实测锚点分段插值（Timucin-Morris 1970 / Suito-Inoue 1984，随碱度）；SiO₂/P₂O₅ 用离子结构理论式。
- 平衡常数 lgK = A/T + B（JSPS/MSTS 推荐值）：C(1160, 2.00)、Si(30110, −11.40)、Mn(12760, −5.68)、P(18355, −14.53, 每 mol P)、Fe(6320, −2.73)。
