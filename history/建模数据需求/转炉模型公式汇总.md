# 转炉冶炼机理模型 · 全部公式汇总（论文公式 + 程序公式两类）

> **用途**：①第一类=**需写入论文**的核心机理/热力学公式，带符号说明与代入示例（代表炉：铁水 136.5 t/1330 ℃，废钢 40 t，供氧 617 m³/min，终点 C 0.049%/T 1630 ℃）；②第二类=**不写入论文、但程序中存在**的公式，用于理解整体架构与**判断可简化/可删除处**。
> **「程序状态」列**（第二类关键）：【引擎在用】实际参与计算 ｜【休眠·未调用】函数已定义但引擎未调用（多为张强原版，可删）｜【已被取代】旧实现被修正版替换 ｜【数值保护】钳位/下界，非物理。
> 公式以 V1.0 冻结代码为准。示例数值为量级演示，精确值以 `scripts/_paper_stats.py` 快照为准。

## 〇、符号约定（全文统一）

| 记号 | 含义 |
|---|---|
| $\mathrm{[M]}$ / $\mathrm{(MO_n)}$ | 钢液中组元 / 渣中组元 |
| 上标 $\mathrm{b}$ / 上标 $*$ | 本体（bulk）/ 渣-金界面 |
| $\mathrm{w(\cdot)}$ | 质量分数，% |
| $\mathrm{a_O^{*}}$ / $\mathrm{a_O^{bulk}}$ | 界面氧活度 / 碳氧缓冲本体氧活度（1% 标准态） |
| $\mathrm{F_M},\ \mathrm{F_{MO_n}}$ | 钢侧/渣侧传质通量因子，$\mathrm{mol\cdot m^{-2}\cdot s^{-1}\cdot \%^{-1}}$ |
| $\mathrm{J_i}$ | 界面摩尔通量，$\mathrm{mol\cdot m^{-2}\cdot s^{-1}}$ |
| $\mathrm{K_M}$ / $\mathrm{E_M}$ | 平衡常数 / 表观平衡常数 |
| $\mathrm{f_i}$ / $\text{γ}_\mathrm{MO}$ | 钢液活度系数（1% 标准态）/ 渣中氧化物 Raoultian 活度系数 |
| $\mathrm{T}$ | 温度，K；成分单位 %；质量 kg；时间 s |

（本文档符号一律正体排印，便于屏读；投稿排版时按期刊规范处理。）

## A. 动力学：混合能与动态传质（model/mixing_energy.py, modified_mass_transfer.py）

| # | 公式 | 符号与单位 | 代入示例 |
|---|---|---|---|
| A1 | $\text{ε}_\mathrm{T}=\dfrac{0.0403\cdot \mathrm{Q_T}\cdot \mathrm{D}\cdot \mathrm{U}^{2}\cdot \cos^{2}\text{θ}}{\mathrm{W}\cdot \mathrm{X}}$ | $\mathrm{Q_T}$ 供氧流量；$\mathrm{D}$ 喷嘴径 m；$\mathrm{U}$ 射流速度（≈300 m/s）；$\text{θ}$ 夹角；$\mathrm{W}$ 熔池质量 kg；$\mathrm{X}$ 枪位 m | $\dfrac{0.0403\cdot 617\cdot 0.042\cdot 300^{2}\cdot \cos^{2}12^{\circ}}{170000\cdot 1.5}\approx 0.35$（模型只用比值 $\text{ε}/\text{ε}^{0}$，量纲随比值消去） |

**A 示例详解**：混合能把工艺参数（供氧、枪位、底吹、熔池几何）折算成搅拌强度，再经 A3/A4 映射到传质系数——这是"不同炉次输入 → 不同动力学"的入口。$\text{γ}$、$\text{α}$ 为设备级标定参数。**论文写法**：$\text{ε}_\mathrm{T}/\text{ε}_\mathrm{B}$ 引 Nakanishi；$\mathrm{k_m}/\mathrm{k_s}$ 引 Kitamura(2009)，并注明本文以 $\mathrm{W_{ref}}$ 作工业尺度归一（取代原 40/m_slag 无量纲因子）。
